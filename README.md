🥳 卡通转盘 (Cartoon Spinner)

一个充满乐趣、视觉效果出色的 iOS 决策工具。

这个应用程序提供了一个生动有趣的转盘界面，旨在帮助用户快速做出决定，或为聚会和日常选择增加乐趣。应用支持多转盘管理和实时数据存储。

✨ 主要功能

多转盘管理： 支持创建、切换和删除多个独立的转盘集合（例如：“今天吃什么？”、“真心话大冒险”）。

高度可定制： 用户可以添加、删除、编辑和重新排序转盘中的选项，并自定义每个扇区的颜色。

沉浸式动画： * 平滑且逼真的旋转物理效果（加速和缓停）。

指针在应用启动或切换转盘时，会自动对准第一个选项的中心。

转盘停止后有🎉 撒花特效和吸睛的结果弹窗。

历史记录： 自动保存所有转盘的结果，方便用户回顾。

现代化 UI： 采用 SwiftUI 编写，界面美观、色彩鲜明，并带有卡片式设计和柔和阴影。

触觉反馈： 旋转和停止时提供细腻的震动反馈。

🛠️ 技术栈

平台: iOS

语言: Swift

框架: SwiftUI

数据存储: 使用 @AppStorage 进行本地持久化存储，保存所有转盘数据和历史记录。

动画/特效: 利用 withAnimation 和 Combine Timer 实现核心旋转和过渡效果。

🚀 如何使用

1. 旋转转盘

打开应用，转盘指针将自动对准第一个选项的中心。

点击转盘中央的 “GO” 按钮开始旋转。

在转盘旋转过程中，再次点击中央的 “STOP” 按钮可以提前停止转动，或者等待约 2.5 秒，它将自动减速并停止。

转盘停止后，会弹出结果弹窗并伴有撒花特效。

2. 管理转盘

切换转盘： 点击顶部中央的转盘名称（如“今天吃什么？”），进入选择器界面。您可以切换到已有的转盘，或点击 “创建新转盘”。

3. 自定义选项和名称 (设置)

点击左上角的 设置（齿轮图标 ⚙️）。

编辑名称： 在顶部的“转盘名称”输入框中修改当前转盘的名称。

管理选项： * 在“选项列表”中，点击任一选项进入编辑页面，您可以修改该选项的文本和扇区颜色。

使用列表上方的 “编辑” 按钮可以对选项进行拖拽排序或删除。

添加选项： 在底部的“添加新选项”输入框中输入内容，然后点击 “添加”。

所有更改都会在您离开设置页面时自动保存。

4. 查看历史记录

点击右上角的 历史记录（时钟图标 🕰️）。

在这里您可以查看所有转出的结果，包括结果名称和所属转盘。

您可以滑动删除单条记录，或点击 “清空” 来移除所有历史记录。
